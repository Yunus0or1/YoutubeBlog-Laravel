A simple laravel project to write blog about any Youtube video's in Bangla

First copy 'YoutubeBlog' folder in your hard drive.
Load 'youtubeBlog.sql' in Xampp database.
Open CMD.
Add the path of the folder in cmd : cd /d D:/routine
Write this command : php artisan serve
Open the link in browser.
